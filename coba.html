Halaman 1 berisi ordered list, gambar, dan tabel yang berisi biodata.

Ordered List

  1. tag html digunakan mendefinisikan sebuah HTML docs halaman 1
  2. tag head digunakan untuk mendefinisikan informasi halaman 1
  3. tag title digunakan untuk memberi judul halaman 1
  4. tag body digunakan untuk mendefiniskan body dari halaman 1
  5. tag ol digunakan untuk membuat ordered list pada halaman 1
  6. tag li digunakan untuk membuat isi dari list pada halaman 1
  7. tag table digunakan untuk membuat tabel
  8. tag th digunakan untuk membuat header dari sebuah tabel
  9. tag tr digunakan untuk membuat baris dari sebuah tabel
  10. tag td digunakan untuk membuat cell pada tabel
  11. tag img digunakan untuk menampilkan gambar
  12. tag br digunakan untuk membuat "single line break" antara list dengan gambar
  13. tag style digunakan untuk membuat style pada halaman 1
  14. tag i digunakan untuk memiringkan tulisan pada list no. 12
  15. tag b digunakan untuk menebalkan tulisan pada heading
  16. tag h1 digunakan untuk membuat heading bertuliskan "first try bismillah"
  17. tag p digunakan untuk menampilkan tulisan pada halaman 1 atau page 1
  18. tag caption digunakan untuk memberi nama tabel, disini diberi nama "BIODATA"
  19. tag div digunakan untuk "memisahkan" bagian ordered list dengan yang lain dengan memberi warna background berbeda
